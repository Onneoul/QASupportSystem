tkinter를 사용하여 다음 조건을 만족하는 코드를 작성해줘
1. mysql에서 게임의 정보를 읽어올 수 있어야해.
게임의 정보는 다음과 같이 구성됨.
1-1. 게임 제목, 1-2. 게임 실행파일(exe) 경로, 1-3. 게임 버전(version), 1-4 로그 경로

2. 창의 화면은 다음과 같이 구성됨.
2-1. DB에 등록된 게임 제목, 버전이 함께 출력되는 ComboBox
2-2. ComboBox 오른쪽에 위치한 추가 버튼(3번에 자세히 설명)
2-3. 창의 최하단에 위치한 실행 버튼, 이를 클릭할 경우 게임 exe 파일이 실행됨.

3. '추가' 버튼을 넣어 이를 클릭할 시 별도의 창을 열어 게임의 정보를 입력할 수 있음.
3-1. 입력하는 게임의 정보는 제목, 실행파일 경로, 게임 버전, 로그 경로로 구성됨.
3-2. 실행파일 경로는 파일 탐색기를 사용하여 실행파일(exe)파일을 선택함.
3-3. 로그 경로도 파일 탐색기를 사용해 특정 폴더를 선택함.
3-4. 이후 '완료' 버튼을 누르면 해당 게임의 정보가 mysql db의 GameInformation 테이블에 추가됨. (SQL은 구성중이므로 SQL문이 들어갈 위치만 알려주면 됨.)


4. 


매크로 연동부분

2-3. ComboBox 아래쪽에 위치한 ComboBox에서 특정 게임 버전을 선택하면 해당 게임 버전에 해당하는 매크로의 목록을 보여주는 ListBox
2-4. 해당 게임 버전에 해당하는 매크로에 대한 정보는 DB에 있으며, SQL은 따로 작성하지 말고 SQL이 위치해야 하는 자리를 알려주면 됨. 매크로에 대한 정보는 매크로 이름, 매크로 경로로 구성.
2-5. 리스트 박스 오른쪽에 위치한 실행 버튼, 이를 클릭할 시 매크로 경로에 위치한 매크로를 읽고 실행시킴. (실행은 다른 프로그램으로 하므로 일단은 자리만 알려주면 됨.)
